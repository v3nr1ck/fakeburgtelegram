# Draft queue (not on the live site)

These stories are **ready for the CMS** but **not published**.

`build.py` and the live site only use `content/articles/`.  
This folder is ignored until you move a file over (or paste into **New article**).

## How to slow-roll one story

### Option A — CMS paste
1. Open the draft `.md` file  
2. Copy **headline** → CMS title  
3. Copy body (everything below the `---` block) → article body  
4. Set section / homepage / photo in the CMS  
5. Save & publish when you want it live  

### Option B — Drop into articles folder
1. Move (or copy) one file from here into `content/articles/`  
2. Open CMS → edit if needed → **Publish**  

Tip: change the `date:` in the frontmatter to “today” when you release it so it sorts to the top.
